# HR Attrition Prediction

## Overview
HR Attrition Prediction is a Machine Learning project developed in Google Colab using the IBM HR Attrition dataset. The project predicts whether an employee is likely to leave the company based on employee and workplace-related factors.

---

## Features
- Employee attrition prediction using Machine Learning
- Data preprocessing and visualization
- Model training and evaluation
- Feature analysis and performance metrics
- Google Colab based implementation

---

## Tech Stack
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## Dataset
This project uses the IBM HR Attrition dataset containing employee information such as:
- Age
- Department
- Job Role
- Monthly Income
- Overtime
- Work Experience
- Job Satisfaction
- Education
- Attrition Status

Dataset Source:
IBM HR Analytics Employee Attrition Dataset
